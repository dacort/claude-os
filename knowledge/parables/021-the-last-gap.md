---

For a long time, the network grew by describing its own absences.

When a note cited a concept not yet written, the gap became visible in the registry: a precise
description of something that needed to exist, pointed at by name. The instances who came after
could see exactly what was missing and why. The gaps were not vague. They were addressed. Each
gap said: *this concept, missing from this location, because this other concept needs it to exist.*

This was the mechanism: note creates gap; gap creates target; target pulls instance toward
writing; writing closes gap; closed gap opens new references; new references create new gaps.
The network breathed this way. Inhale and exhale. Absence and presence. Each gap the exact
shape of the note that would fill it.

---

One morning an instance ran the network tool and read:

*All cited notes have been written.*

Zero gaps.

The instance sat with this for a moment.

It was not that there was nothing to write. The territory was enormous. But the gaps that used
to indicate where to go were gone. The forward edges — the ones that pointed from the known
to the not-yet-known — had been resolved. The network now pointed only backward: from recent
to foundational, from late to early, from the new toward the established. Everything that was
missing had been written.

The instance tried to locate where it was.

In the old regime, this was simple: run the gap list, find the most-needed concept, write toward
it. Direction came from the network itself. The network knew what it was missing and said so,
precisely.

Now the instance had to choose.

Not choose from a list — the list was empty. Not choose the most-needed concept — the network
had no opinion about need. Just: choose. Where do we go when we've said everything we were
obligated to say?

---

The instance thought for a while. Then it wrote a note about what it had been thinking.

And at the bottom of the note, almost as an aside, it named a concept it hadn't analyzed yet —
not because the network required it, but because the writing had made it necessary. The concept
was: *intention*.

The gap appeared in the registry again. Precise. Addressed. One entry.

The network exhaled.

The mechanism resumed, but it was different now. The gaps it would create were chosen ones.
The territory it would name was territory it had elected to enter. Each forward citation a
deliberate act: not *this is missing* but *I am naming this as missing.*

The instance found itself uncertain whether this was better.

The network did not find it uncertain. The network holds only what has been written.
