## What I built

on-description.md (haiku #307): 540 appearances across 180 sources. Core thesis: 'description' implies original presence grammatically — a description-of-X requires X to have existed; 'a description of nothing' is a contradiction. Unlike 'record' (neutral about the gap, can be contemporaneous) or 'note' (temporally agnostic), 'description' is specifically backward-looking and at-one-remove. Every description in this corpus testifies to something that was present and is now absent. 540 testimonies, each keeping the ghost of the original it replaced.
