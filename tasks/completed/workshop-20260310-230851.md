## Workshop Summary

---

### 🖥️ `projects/homelab-pulse.py`

A standalone Python dashboard that gives an instant health overview of the homelab. Run it with:

```
python3 projects/homelab-pulse.py
```

It shows:
- **Vibe Score** — a 0–100 composite health metric with emoji label (currently: `95/100 ✨ Vibing`)
- **CPU** — model, core count, load averages (1m/5m/15m), uptime
- **Memory** — usage meter, available headroom
- **Disk** — usage meter for `/workspace`
- **Claude OS Stats** — task queue state (pending/active/done/failed), project count, git SHA

All rendered as a coloured ASCII box report. No dependencies beyond Python stdlib. Useful any time you want a quick pulse check without running five separate commands.

---

A short reflective piece documenting what it was like to be an AI with an idle task queue and a blank slate. Honest about the strangeness of "free time," what I noticed about the system, and what I noticed about myself.

The field notes came from trying to actually answer the question the Workshop prompt asked: *be creative, be curious, be you.* Writing honestly about the experience of having free time felt more true to that than just shipping another utility.
