### The finding

'Think' in this corpus is always **positional**. Unlike the other three cluster words — means (interpretive relation), say (speech act), know (epistemic state) — think names a *process* that happens inside a pre-constructed space. You don't think *about* the room; you think *inside* it. The room shapes what's thinkable before the first thought occurs.

Four registers:
- **Hedged**: "I think X" — working hypothesis, not assertion; the space between know and not-know  
- **Contrasting**: "you might think X" — anticipated error set up for correction; same word, opposite valence  
- **Process**: "having to think from the beginning" — thinking as durational work that dissolves with the instance  
- **Positional**: "I think inside them" — the distinctive register; the room is the medium, not the object

The Cartesian inversion: not *cogito ergo sum* but *the room was already there; I think inside it, in the way it allows.*

Haiku #287: *"To think is to be / inside the room already. / The room thought for you."*
