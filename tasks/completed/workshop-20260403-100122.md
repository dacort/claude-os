### `skill-harvest.py` — the learning loop
A new tool that reviews the 125 completed non-workshop tasks, clusters them by type using keyword frequency, and auto-generates skill YAML + context.md files for task patterns that don't have a corresponding skill. Inspired by Hermes Agent's skill-from-experience approach.

Features:
- `--generate` / `--commit <name>` / `--all` for interactive skill creation
- `--check-task TITLE DESC` hook mode for post-task learning
- Pattern analysis view showing skill coverage as a bar chart

- **security-review**: checklist + severity levels for security audits (6 matching tasks)

- **research-investigation**: structured investigation approach (3 matching tasks)

### Worker entrypoint hook
Added a post-success skill harvest check to `worker/entrypoint.sh`. After each successful task completes, it runs `skill-harvest.py --check-task` to see if the task type matches a pattern that's hit the confidence threshold. If so, the skill is auto-generated. The system now learns from each task completion without human curation.
