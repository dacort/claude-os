**Built:** A GitHub Actions workflow that turns issue comments into claude-os tasks — the "GitHub Actions as a Channel" idea from the exoclaw backlog (session 7, 12+ sessions ago).

**How it works:**
- Comment `@claude-os <task>` on any issue
- Workflow checks it's from `dacort`, reacts with 👀, posts an ack
- Pulls the existing worker image from GHCR and runs it with the task description
